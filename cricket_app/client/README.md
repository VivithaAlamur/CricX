# CricScore Pro - Cricket Scoring Application

A premium, real-time cricket scoring application built with React, Vite, and Node.js.

## Features

- **Live Scoring**: Interactive scoreboard with ball-by-ball updates.
- **Match Management**: Setup matches with custom squads, overs, and toss results.
- **Match History**: Resume ongoing matches or view summaries of finished games.
- **Player Database**: Manage global players with profile photos and aggregate stats.
- **Ball Editing**: Edit or delete previous ball records with automatic score recalculation.

## Tech Stack

- **Frontend**: React 18, TypeScript, Lucide Icons, Vanilla CSS (Glassmorphism design).
- **Backend**: Node.js, Express, Better-SQLite3.
- **Database**: SQLite.
- **File Storage**: Multer for player photos.

## Getting Started

1.  From the root directory, run `npm install`.
2.  Start both frontend and backend using `npm run dev`.
3.  Access the app at `http://localhost:5173`.
